News Sift Screens

Landing Page
    User Flows:
  • User can click on get started now to be taken to the registration page
    • Registration Page
      User Flows:
  • user enters a valid user name and password, and selects at least one preferred source. => new account is created and user is directed to personal feed
  • user enters invalid account data (missing required fields, username taken, not selecting at least one source) => user is prompted to fix issues before continuing
    • Search Page
      User Flows:
  • user enters search term, chooses sort and display method and submits search. => results from the users preferred sources are provided.
  • user clicks on article title or image. => user is redirected to the article url.
  • user clicks on publication name. => user is directed to publication homepage.
    • Feed Page
      User Flows:
  • user clicks on check the news. new get request is made to return updated results.
  • user clicks on article title or image. => user is redirected to the article url.
  • user clicks on publication name. => user is directed to publication homepage.
  • user clicks on see more. the next set of articles from the original get request are shown.
    • Footer
